# Bike Purchase Analysis Dashboard (Excel)

## Overview
This project is an interactive Excel dashboard designed to analyze customer characteristics and factors influencing bike purchase decisions. The dashboard uses Pivot Tables, Pivot Charts, and Slicers to provide dynamic insights into customer behavior.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning and Preparation

## Dashboard Features

### Interactive Filters
- Marital Status
- Region
- Education Level

### Analysis Performed
- Average Income by Gender and Bike Purchase Status
- Customer Commute Distance Analysis
- Age Bracket Analysis
- Regional Purchase Trends
- Demographic-Based Purchase Behavior

## Key Insights
- Customers with higher average income were more likely to purchase bikes.
- Commute distance had a noticeable impact on purchasing decisions.
- Middle-aged customers showed stronger purchasing tendencies.
- Purchase behavior varied across regions and education levels.
- Customer demographics played a significant role in bike purchasing patterns.
